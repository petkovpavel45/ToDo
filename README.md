# ToDo App

A clean, minimal task manager built with React.

**[Live Demo →](https://todo-vite-kxxigipi3-vibe-coding4.vercel.app)**

---

## Features

- **Add tasks** — type and press Enter or click +
- **Complete tasks** — click the circle to toggle done/undone
- **Delete tasks** — remove individual tasks with ✕
- **Filter** — All / Active / Done
- **Task counter** — shows remaining tasks
- **Clear completed** — remove all done tasks at once
- **LocalStorage** — tasks persist after page refresh
